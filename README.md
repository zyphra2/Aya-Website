# Aya Es-samlaly — Personal Site

Static portfolio site scaffold. Open `index.html` in a browser to view.

Files:
- `index.html` — main page with 3D hero background, About, Contact.
- `styles.css` — styles, neon/glow accents and responsive layout.
- `script.js` — Three.js animated 3D background (torus knot) with red/blue neon lights.

How to run:
1. Open `index.html` in a modern browser (Chrome/Edge/Firefox). No build step required.
2. To serve locally (recommended) using Python: `python -m http.server 8000` then open `http://localhost:8000`.
